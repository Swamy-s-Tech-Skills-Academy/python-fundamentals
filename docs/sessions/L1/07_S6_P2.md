## 5) Essential Built-in Functions (Canonical Scope)

| Function | Why it matters |
| --- | --- |
| `len()` | Count items/chars for bounds checks and summaries |
| `max()` | Find the largest value quickly |
| `min()` | Find the smallest value quickly |
| `sum()` | Aggregate totals for reporting/calculations |
| `abs()` | Measure distance from zero (difference magnitude) |
| `round()` | Format numeric output to practical precision |

**Reasoning:** These six built-ins cover the most common beginner data checks and summaries without introducing extra complexity.

## 6) How to Apply Built-ins in Practice

- Combine `sum()` and `len()` for averages.
- Use `max()` and `min()` to detect ranges/outliers.
- Use `abs()` for difference-based comparisons.
- Use `round()` for display-friendly numeric results.

## 7) Troubleshooting Quick Guide

- `SyntaxError` / `IndentationError` → fix structure first
- `NameError` → verify exact variable names
- `TypeError` / `ValueError` → validate and convert input early
- `IndexError` / `KeyError` → check existence before access
- `ZeroDivisionError` → guard divisor before dividing
