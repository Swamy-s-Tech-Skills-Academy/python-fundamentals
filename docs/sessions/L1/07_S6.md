## ‚ùå **Part 1: Understanding Errors**

### **Two Types of Errors**

| Error Type      | When It Happens                    | Example                          |
| --------------- | ---------------------------------- | -------------------------------- |
| **Syntax Error**| Before code runs (compilation)     | Missing colon, wrong indentation |
| **Runtime Error**| While code is running             | Division by zero, wrong type     |

### **Syntax Errors**

Syntax errors occur when Python can't understand your code. The code doesn't run at all!

```python
# Syntax Error Examples

# Missing colon
if True
    print("Hello")  # SyntaxError: expected ':'

# Wrong indentation
if True:
print("Hello")  # IndentationError: expected an indented block

# Mismatched quotes
print("Hello')  # SyntaxError: unterminated string literal

# Missing parenthesis
print("Hello"  # SyntaxError: '(' was never closed
```

### **Runtime Errors (Exceptions)**

Runtime errors occur while the code is running. The code starts but crashes at a specific point.

```python
# Runtime Error Examples

# Division by zero
result = 10 / 0  # ZeroDivisionError: division by zero

# Wrong type operation
"hello" + 5  # TypeError: can only concatenate str (not "int") to str

# Accessing non-existent index
my_list = [1, 2, 3]
print(my_list[10])  # IndexError: list index out of range

# Using undefined variable
print(undefined_var)  # NameError: name 'undefined_var' is not defined
```

### **Common Exception Types**

| Exception          | Cause                                      |
| ------------------ | ------------------------------------------ |
| `SyntaxError`      | Invalid Python syntax                      |
| `IndentationError` | Wrong indentation                          |
| `NameError`        | Variable or function not defined           |
| `TypeError`        | Wrong type for an operation                |
| `ValueError`       | Right type but wrong value                 |
| `IndexError`       | List/string index out of range             |
| `KeyError`         | Dictionary key not found                   |
| `ZeroDivisionError`| Division by zero                           |

## üìñ **Part 2: Reading Error Messages**

### **Anatomy of a Traceback**

When an error occurs, Python shows a **traceback** with helpful information:

```text
Traceback (most recent call last):
  File "example.py", line 5, in <module>
    result = 10 / 0
             ~~~^~~
ZeroDivisionError: division by zero
```

**Breaking it down:**

1. **`Traceback (most recent call last):`** - Error occurred, details follow
2. **`File "example.py", line 5`** - Which file and line number
3. **`result = 10 / 0`** - The actual code that caused the error
4. **`ZeroDivisionError: division by zero`** - Error type and description

### **Reading Multiple-Level Tracebacks**

```python
# File: main.py
def divide(a, b):
    return a / b

def calculate():
    result = divide(10, 0)
    return result

calculate()
```

**Traceback:**

```text
Traceback (most recent call last):
  File "main.py", line 9, in <module>
    calculate()
  File "main.py", line 6, in calculate
    result = divide(10, 0)
  File "main.py", line 3, in divide
    return a / b
           ~~^~~
ZeroDivisionError: division by zero
```

**Reading order:** Start from the BOTTOM! The last entry shows where the error actually occurred.

### **Tips for Reading Errors**

1. **Look at the error type first** - What kind of error is it?
2. **Read the error message** - What specifically went wrong?
3. **Find the line number** - Where in your code?
4. **Look at the code shown** - What's on that line?
5. **Check the context** - What's happening around that code?

- [ ] Identify the error type from a traceback
- [ ] Find the line number where the error occurred

## üêõ **Part 3: Common Beginner Mistakes**

### **Mistake 1: Forgetting Colons**

```python
# Wrong
if x > 5
    print("Greater")

# Right
if x > 5:
    print("Greater")
```

### **Mistake 2: Wrong Indentation**

```python
# Wrong
if True:
print("Hello")
    print("World")

# Right
if True:
    print("Hello")
    print("World")
```

### **Mistake 3: Using = Instead of ==**

```python
# Wrong (assignment, not comparison)
if x = 5:
    print("Five")

# Right (comparison)
if x == 5:
    print("Five")
```

### **Mistake 4: Forgetting Parentheses**

```python
# Wrong
print "Hello"

# Right
print("Hello")
```

### **Mistake 5: String/Number Confusion**

```python
# Wrong
age = input("Age: ")
if age > 18:  # TypeError! age is a string
    print("Adult")

# Right
age = int(input("Age: "))
if age > 18:
    print("Adult")
```

### **Mistake 6: Modifying While Iterating**

```python
# Problematic
numbers = [1, 2, 3, 4, 5]
for num in numbers:
    if num % 2 == 0:
        numbers.remove(num)  # Don't modify while iterating!

# Better
numbers = [1, 2, 3, 4, 5]
numbers = [num for num in numbers if num % 2 != 0]
```

### **Mistake 7: Off-by-One Errors**

```python
# Prints 0-4, not 1-5
for i in range(5):
    print(i)

# Prints 1-5
for i in range(1, 6):
    print(i)
```

### **Mistake 8: Infinite Loops**

```python
# Wrong - infinite loop!
count = 0
while count < 5:
    print(count)
    # Forgot: count += 1

# Right
count = 0
while count < 5:
    print(count)
    count += 1
```

## üîç **Part 4: Debugging with print()**

### **Print Debugging Strategy**

When your code doesn't work as expected, add `print()` statements to see what's happening:

```python
def calculate_discount(price, discount_percent):
    print(f"DEBUG: price = {price}")  # Debug line
    print(f"DEBUG: discount = {discount_percent}")  # Debug line
    
    discount_amount = price * discount_percent / 100
    print(f"DEBUG: discount_amount = {discount_amount}")  # Debug line
    
    final_price = price - discount_amount
    print(f"DEBUG: final_price = {final_price}")  # Debug line
    
    return final_price

# Test
result = calculate_discount(100, 20)
print(f"Result: {result}")
```

### **What to Print**

1. **Variable values** - What's actually stored?
2. **Loop iterations** - How many times does the loop run?
3. **Condition results** - Which branch is taken?
4. **Function inputs/outputs** - What goes in and comes out?

### **Debugging Loop Example**

```python
# Something's wrong with this code...
numbers = [1, 2, 3, 4, 5]
total = 0

for i in range(len(numbers)):
    print(f"DEBUG: i = {i}, numbers[i] = {numbers[i]}")  # Debug
    total += numbers[i]
    print(f"DEBUG: total so far = {total}")  # Debug

print(f"Final total: {total}")
```

### **Remove Debug Prints When Done**

```python
# Clean code after debugging
numbers = [1, 2, 3, 4, 5]
total = 0

for num in numbers:
    total += num

print(f"Total: {total}")
```

### **üí° Pro Tip: Use DEBUG Prefix**

Always prefix debug prints with "DEBUG:" so they're easy to find and remove later!

## üõ†Ô∏è **Part 5: Essential Built-in Functions**

### **Functions Covered**

| Function  | Purpose                              | Example                    |
| --------- | ------------------------------------ | -------------------------- |
| `len()`   | Get length of sequence               | `len("hello")` ‚Üí `5`       |
| `max()`   | Find maximum value                   | `max(1, 5, 3)` ‚Üí `5`       |
| `min()`   | Find minimum value                   | `min(1, 5, 3)` ‚Üí `1`       |
| `sum()`   | Sum all items                        | `sum([1, 2, 3])` ‚Üí `6`     |
| `abs()`   | Absolute value                       | `abs(-5)` ‚Üí `5`            |
| `round()` | Round a number                       | `round(3.7)` ‚Üí `4`         |

### **len() - Get Length**

```python
# String length
text = "Hello, World!"
print(len(text))  # Output: 13

# List length
fruits = ["apple", "banana", "cherry"]
print(len(fruits))  # Output: 3

# Empty check
if len(fruits) > 0:
    print("List has items")
```

### **max() and min() - Find Extremes**

```python
# With individual arguments
print(max(10, 5, 8, 3))  # Output: 10
print(min(10, 5, 8, 3))  # Output: 3

# With a list
numbers = [42, 17, 89, 23, 56]
print(f"Highest: {max(numbers)}")  # Output: 89
print(f"Lowest: {min(numbers)}")   # Output: 17

# With strings (alphabetical)
words = ["apple", "zebra", "mango"]
print(max(words))  # Output: zebra
print(min(words))  # Output: apple
```

### **sum() - Add All Items**

```python
# Sum a list
numbers = [10, 20, 30, 40]
total = sum(numbers)
print(f"Total: {total}")  # Output: 100

# Sum with starting value
total_with_start = sum(numbers, 100)
print(f"Total + 100: {total_with_start}")  # Output: 200

# Calculate average
average = sum(numbers) / len(numbers)
print(f"Average: {average}")  # Output: 25.0
```

### **abs() - Absolute Value**

```python
# Remove negative sign
print(abs(-10))   # Output: 10
print(abs(10))    # Output: 10
print(abs(-3.14)) # Output: 3.14

# Useful for distance calculation
target = 50
guess = 43
distance = abs(target - guess)
print(f"You're {distance} away!")  # Output: You're 7 away!
```

### **round() - Round Numbers**

```python
# Round to nearest integer
print(round(3.7))   # Output: 4
print(round(3.2))   # Output: 3
print(round(3.5))   # Output: 4 (banker's rounding)

# Round to specific decimal places
pi = 3.14159265
print(round(pi, 2))  # Output: 3.14
print(round(pi, 4))  # Output: 3.1416

# Useful for money
price = 19.9999
print(f"Price: ${round(price, 2)}")  # Output: $20.0
```

### **Combining Built-in Functions**

```python
scores = [85, 92, 78, 96, 88, 73, 91]

print(f"Number of scores: {len(scores)}")
print(f"Highest score: {max(scores)}")
print(f"Lowest score: {min(scores)}")
print(f"Total points: {sum(scores)}")
print(f"Average: {round(sum(scores) / len(scores), 2)}")
print(f"Range: {max(scores) - min(scores)}")
```
