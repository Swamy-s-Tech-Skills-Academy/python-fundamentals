# Session 6: Basic Debugging & Built-in Functions

**Duration:** 30 minutes  
**Type:** 📚 Knowledge  
**Level:** Noob → Nerd

---

## 🎯 Session Overview

In this session, you build a reliable beginner debugging process and practice core built-in functions used in real scripts.

### What you'll learn

- ✅ Syntax errors vs runtime errors
- ✅ How to read traceback messages quickly
- ✅ Common beginner mistakes and practical fixes
- ✅ Print-debugging workflow
- ✅ Built-ins: `len()`, `max()`, `min()`, `sum()`, `abs()`, `round()`

### Suggested 30-minute flow

- **0–5 min:** Error types and why they happen
- **5–12 min:** Traceback reading strategy
- **12–18 min:** Common mistakes and print-debugging
- **18–25 min:** Built-in function usage
- **25–30 min:** Run and review practice scripts

## ✅ Exit Check

By the end of Session 6, you should be able to:

- Identify whether an error is syntax or runtime.
- Locate and explain the failing line in a traceback.
- Add targeted debug prints, then remove them after fixing.
- Use `len`, `max`, `min`, `sum`, `abs`, and `round` in small scripts confidently.
