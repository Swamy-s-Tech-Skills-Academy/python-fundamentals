## üî¢ **Part 1: Arithmetic Operators**

### **What are Arithmetic Operators?**

Arithmetic operators perform mathematical calculations. Python supports all the standard math operations you learned in school ‚Äî plus a few extras!

### **The Seven Arithmetic Operators**

| Operator | Name             | Example    | Result |
| -------- | ---------------- | ---------- | ------ |
| `+`      | Addition         | `5 + 3`    | `8`    |
| `-`      | Subtraction      | `10 - 4`   | `6`    |
| `*`      | Multiplication   | `6 * 7`    | `42`   |
| `/`      | Division         | `15 / 4`   | `3.75` |
| `//`     | Floor Division   | `15 // 4`  | `3`    |
| `%`      | Modulo (Remainder)| `15 % 4`  | `3`    |
| `**`     | Exponentiation   | `2 ** 3`   | `8`    |

### **Basic Arithmetic Examples**

```python
# Addition and Subtraction
print(10 + 5)   # Output: 15
print(10 - 5)   # Output: 5

# Multiplication
print(6 * 7)    # Output: 42

# Division (always returns a float)
print(15 / 4)   # Output: 3.75
print(10 / 2)   # Output: 5.0 (still a float!)
```

### **Floor Division vs Regular Division**

**Floor division (`//`)** divides and rounds DOWN to the nearest whole number:

```python
print(15 / 4)   # Output: 3.75 (regular division)
print(15 // 4)  # Output: 3    (floor division - rounds down)

print(17 / 5)   # Output: 3.4
print(17 // 5)  # Output: 3

# Works with negative numbers too (rounds toward negative infinity)
print(-15 // 4) # Output: -4 (not -3!)
```

### **Modulo Operator (Remainder)**

The **modulo operator (`%`)** gives you the remainder after division:

```python
print(15 % 4)   # Output: 3 (15 √∑ 4 = 3 remainder 3)
print(10 % 3)   # Output: 1 (10 √∑ 3 = 3 remainder 1)
print(20 % 5)   # Output: 0 (20 √∑ 5 = 4 remainder 0)
```

**Common uses for modulo:**

```python
# Check if a number is even or odd
number = 7
print(number % 2)  # Output: 1 (odd numbers give 1, even give 0)

# Check if divisible
print(15 % 5 == 0)  # Output: True (15 is divisible by 5)
print(15 % 4 == 0)  # Output: False (15 is not divisible by 4)
```

### **Exponentiation (Power)**

The **exponentiation operator (`**`)** raises a number to a power:

```python
print(2 ** 3)   # Output: 8  (2 √ó 2 √ó 2)
print(5 ** 2)   # Output: 25 (5 √ó 5)
print(10 ** 0)  # Output: 1  (any number to power 0 is 1)

# Square root using exponentiation
print(16 ** 0.5)  # Output: 4.0 (square root of 16)
print(27 ** (1/3)) # Output: 3.0 (cube root of 27)
```

## ‚öñÔ∏è **Part 2: Comparison Operators**

### **What are Comparison Operators?**

Comparison operators compare two values and return a **Boolean** (`True` or `False`). These are essential for making decisions in your code!

### **The Six Comparison Operators**

| Operator | Meaning                  | Example    | Result  |
| -------- | ------------------------ | ---------- | ------- |
| `==`     | Equal to                 | `5 == 5`   | `True`  |
| `!=`     | Not equal to             | `5 != 3`   | `True`  |
| `<`      | Less than                | `3 < 5`    | `True`  |
| `>`      | Greater than             | `5 > 3`    | `True`  |
| `<=`     | Less than or equal to    | `5 <= 5`   | `True`  |
| `>=`     | Greater than or equal to | `5 >= 3`   | `True`  |

### **Comparison Examples**

```python
# Equality comparisons
print(5 == 5)    # Output: True
print(5 == 3)    # Output: False
print("hello" == "hello")  # Output: True
print("Hello" == "hello")  # Output: False (case-sensitive!)

# Not equal
print(5 != 3)    # Output: True
print(5 != 5)    # Output: False

# Greater than / Less than
print(10 > 5)    # Output: True
print(10 < 5)    # Output: False

# Greater than or equal / Less than or equal
print(5 >= 5)    # Output: True
print(5 <= 10)   # Output: True
```

### **‚ö†Ô∏è Common Mistake: `=` vs `==`**

```python
# WRONG - This assigns a value, doesn't compare!
# x = 5  (assignment)

# RIGHT - This compares values
# x == 5  (comparison, returns True or False)

x = 5
print(x == 5)   # Output: True (comparison)
print(x == 10)  # Output: False (comparison)
```

### **Comparing Different Types**

```python
# Numbers can be compared across int and float
print(5 == 5.0)     # Output: True
print(10 > 9.5)     # Output: True

# Strings are compared alphabetically (lexicographically)
print("apple" < "banana")  # Output: True (a comes before b)
print("abc" < "abd")       # Output: True
```

### **Chained Comparisons**

Python allows you to chain comparisons elegantly:

```python
x = 5
# Instead of: x > 0 and x < 10
print(0 < x < 10)   # Output: True

age = 25
print(18 <= age <= 65)  # Output: True (working age)
```

## üìù **Part 3: Assignment Operators**

### **What are Assignment Operators?**

Assignment operators are shortcuts for updating variable values. Instead of writing `x = x + 5`, you can write `x += 5`.

### **Common Assignment Operators**

| Operator | Example    | Equivalent To   | Description           |
| -------- | ---------- | --------------- | --------------------- |
| `=`      | `x = 5`    | `x = 5`         | Assign value          |
| `+=`     | `x += 3`   | `x = x + 3`     | Add and assign        |
| `-=`     | `x -= 3`   | `x = x - 3`     | Subtract and assign   |
| `*=`     | `x *= 3`   | `x = x * 3`     | Multiply and assign   |
| `/=`     | `x /= 3`   | `x = x / 3`     | Divide and assign     |
| `//=`    | `x //= 3`  | `x = x // 3`    | Floor divide & assign |
| `%=`     | `x %= 3`   | `x = x % 3`     | Modulo and assign     |
| `**=`    | `x **= 3`  | `x = x ** 3`    | Power and assign      |

### **Assignment Operator Examples**

```python
# Start with a value
score = 100

# Add points
score += 10      # score is now 110
print(score)

# Subtract points
score -= 25      # score is now 85
print(score)

# Double the score
score *= 2       # score is now 170
print(score)

# Halve the score
score /= 2       # score is now 85.0 (becomes float)
print(score)
```

### **Why Use Assignment Operators?**

```python
# Without assignment operators (verbose)
counter = 0
counter = counter + 1
counter = counter + 1
counter = counter + 1
print(counter)  # Output: 3

# With assignment operators (cleaner)
counter = 0
counter += 1
counter += 1
counter += 1
print(counter)  # Output: 3
```

### **String Concatenation with `+=`**

```python
message = "Hello"
message += " "       # message is now "Hello "
message += "World"   # message is now "Hello World"
message += "!"       # message is now "Hello World!"
print(message)       # Output: Hello World!
```

## üéØ **Part 4: Operator Precedence**

### **What is Operator Precedence?**

When an expression has multiple operators, Python follows specific rules about which operations to perform first. This is called **operator precedence** (or "order of operations").

### **Precedence Order (Highest to Lowest)**

| Priority | Operator(s)          | Description                    |
| -------- | -------------------- | ------------------------------ |
| 1        | `()`                 | Parentheses (always first)     |
| 2        | `**`                 | Exponentiation                 |
| 3        | `*`, `/`, `//`, `%`  | Multiplication, Division       |
| 4        | `+`, `-`             | Addition, Subtraction          |
| 5        | `==`, `!=`, `<`, etc.| Comparison operators           |

### **Precedence Examples**

```python
# Multiplication before addition
print(2 + 3 * 4)     # Output: 14 (not 20!)
# Calculated as: 2 + (3 * 4) = 2 + 12 = 14

# Exponentiation before multiplication
print(2 * 3 ** 2)    # Output: 18
# Calculated as: 2 * (3 ** 2) = 2 * 9 = 18

# Division before subtraction
print(10 - 6 / 2)    # Output: 7.0
# Calculated as: 10 - (6 / 2) = 10 - 3.0 = 7.0
```

### **Using Parentheses to Control Order**

Parentheses override the default precedence:

```python
# Without parentheses
print(2 + 3 * 4)     # Output: 14

# With parentheses - force addition first
print((2 + 3) * 4)   # Output: 20

# Complex example
print(10 - 6 / 2)    # Output: 7.0
print((10 - 6) / 2)  # Output: 2.0
```

### **üí° Best Practice: Use Parentheses for Clarity**

Even when not required, parentheses make your code easier to read:

```python
# Hard to read
result = 2 + 3 * 4 - 5 / 2

# Easier to understand
result = 2 + (3 * 4) - (5 / 2)
```

### **Common Formula Examples**

```python
# Calculate average
num1, num2, num3 = 85, 90, 78
average = (num1 + num2 + num3) / 3
print(f"Average: {average}")  # Output: Average: 84.33...

# Calculate area of a circle (œÄ √ó r¬≤)
import math
radius = 5
area = math.pi * (radius ** 2)
print(f"Area: {area:.2f}")  # Output: Area: 78.54

# Temperature conversion (Celsius to Fahrenheit)
celsius = 25
fahrenheit = (celsius * 9/5) + 32
print(f"{celsius}¬∞C = {fahrenheit}¬∞F")  # Output: 25¬∞C = 77.0¬∞F
```

- [ ] Parentheses always execute first

### **Common Issues:**

#### **Division always returns float:**

```python

```

