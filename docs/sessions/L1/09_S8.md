## ğŸ“– **Part 1: Creating Dictionaries**

### **What is a Dictionary?**

A **dictionary** stores data in **key-value pairs**. Think of it like a real dictionary where you look up a word (key) to find its definition (value).

```python
# A simple dictionary
person = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}

# Key: "name" â†’ Value: "Alice"
# Key: "age" â†’ Value: 25
# Key: "city" â†’ Value: "New York"
```

### **Dictionary Characteristics**

- **Unordered** (in older Python) / **Ordered** (Python 3.7+)
- **Mutable** - You can change, add, or remove items
- **Keys must be unique** - No duplicate keys
- **Keys must be immutable** - Strings, numbers, tuples (not lists)
- **Values can be anything** - Any data type

### **Creating Dictionaries**

```python
# Method 1: Curly braces with key-value pairs
student = {
    "name": "Bob",
    "grade": 95,
    "subjects": ["Math", "Science"]
}

# Method 2: dict() constructor
person = dict(name="Charlie", age=30, city="Boston")

# Method 3: Empty dictionary
empty = {}
# or
empty = dict()

# Method 4: From lists of tuples
pairs = [("a", 1), ("b", 2), ("c", 3)]
my_dict = dict(pairs)
print(my_dict)  # {'a': 1, 'b': 2, 'c': 3}
```

### **Dictionary vs List**

| Feature      | List                    | Dictionary                |
| ------------ | ----------------------- | ------------------------- |
| Access by    | Index (position)        | Key (name)                |
| Syntax       | `my_list[0]`            | `my_dict["key"]`          |
| Order        | Always ordered          | Ordered (Python 3.7+)     |
| Best for     | Sequential data         | Labeled/named data        |

## ğŸ”‘ **Part 2: Accessing & Modifying**

### **Accessing Values**

```python
person = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}

# Method 1: Square brackets (raises KeyError if missing)
print(person["name"])  # Alice
print(person["age"])   # 25

# Method 2: get() method (returns None if missing, or default)
print(person.get("name"))        # Alice
print(person.get("country"))     # None (no error!)
print(person.get("country", "Unknown"))  # Unknown (default)
```

### **Adding & Modifying Values**

```python
person = {"name": "Alice", "age": 25}

# Add new key-value pair
person["city"] = "New York"
print(person)  # {'name': 'Alice', 'age': 25, 'city': 'New York'}

# Modify existing value
person["age"] = 26
print(person)  # {'name': 'Alice', 'age': 26, 'city': 'New York'}

# Update multiple values at once
person.update({"age": 27, "country": "USA"})
print(person)  # {'name': 'Alice', 'age': 27, 'city': 'New York', 'country': 'USA'}
```

### **Removing Items**

```python
person = {"name": "Alice", "age": 25, "city": "New York"}

# del - Remove specific key
del person["city"]
print(person)  # {'name': 'Alice', 'age': 25}

# pop() - Remove and return value
age = person.pop("age")
print(age)     # 25
print(person)  # {'name': 'Alice'}

# pop() with default (no error if missing)
country = person.pop("country", "Not found")
print(country)  # Not found

# clear() - Remove all items
person.clear()
print(person)  # {}
```

### **Checking for Keys**

```python
person = {"name": "Alice", "age": 25}

# Check if key exists
print("name" in person)     # True
print("city" in person)     # False
print("city" not in person) # True

# Safe access pattern
if "city" in person:
    print(person["city"])
else:
    print("City not found")
```

## ğŸ› ï¸ **Part 3: Dictionary Methods**

### **Getting Keys, Values, and Items**

```python
person = {"name": "Alice", "age": 25, "city": "New York"}

# keys() - Get all keys
print(person.keys())   # dict_keys(['name', 'age', 'city'])

# values() - Get all values
print(person.values()) # dict_values(['Alice', 25, 'New York'])

# items() - Get all key-value pairs as tuples
print(person.items())  # dict_items([('name', 'Alice'), ('age', 25), ('city', 'New York')])

# Convert to lists if needed
keys_list = list(person.keys())
print(keys_list)  # ['name', 'age', 'city']
```

### **Useful Dictionary Methods**

```python
# setdefault() - Get value, set if missing
person = {"name": "Alice"}
age = person.setdefault("age", 0)  # Sets "age": 0 since missing
print(age)     # 0
print(person)  # {'name': 'Alice', 'age': 0}

name = person.setdefault("name", "Unknown")  # Already exists
print(name)    # Alice (not changed)

# copy() - Create shallow copy
original = {"a": 1, "b": 2}
copy = original.copy()
copy["c"] = 3
print(original)  # {'a': 1, 'b': 2} (unchanged)
print(copy)      # {'a': 1, 'b': 2, 'c': 3}

# fromkeys() - Create dict with same value
keys = ["a", "b", "c"]
zeros = dict.fromkeys(keys, 0)
print(zeros)  # {'a': 0, 'b': 0, 'c': 0}
```

### **Quick Reference: Dictionary Methods**

| Method         | Description                              | Returns        |
| -------------- | ---------------------------------------- | -------------- |
| `get(k, d)`    | Get value for key, or default            | Value          |
| `keys()`       | Get all keys                             | dict_keys      |
| `values()`     | Get all values                           | dict_values    |
| `items()`      | Get all key-value pairs                  | dict_items     |
| `pop(k, d)`    | Remove key and return value              | Value          |
| `update(d)`    | Update with another dict                 | None           |
| `setdefault()` | Get value, set if missing                | Value          |
| `copy()`       | Create shallow copy                      | New dict       |
| `clear()`      | Remove all items                         | None           |

## ğŸ”„ **Part 4: Iterating Over Dictionaries**

### **Iterating Over Keys**

```python
person = {"name": "Alice", "age": 25, "city": "New York"}

# Default iteration (keys)
for key in person:
    print(key)
# Output: name, age, city

# Explicit keys()
for key in person.keys():
    print(f"{key}: {person[key]}")
```

### **Iterating Over Values**

```python
person = {"name": "Alice", "age": 25, "city": "New York"}

for value in person.values():
    print(value)
# Output: Alice, 25, New York
```

### **Iterating Over Items (Key + Value)**

```python
person = {"name": "Alice", "age": 25, "city": "New York"}

# Most common pattern
for key, value in person.items():
    print(f"{key}: {value}")
# Output:
# name: Alice
# age: 25
# city: New York
```

### **Practical Examples**

```python
# Student grades
grades = {"Alice": 95, "Bob": 87, "Charlie": 92}

# Calculate average
total = sum(grades.values())
average = total / len(grades)
print(f"Average: {average:.1f}")  # Average: 91.3

# Find highest grade
highest_name = ""
highest_grade = 0
for name, grade in grades.items():
    if grade > highest_grade:
        highest_grade = grade
        highest_name = name
print(f"Highest: {highest_name} ({highest_grade})")

# Using max() with a key function
best = max(grades, key=grades.get)
print(f"Best student: {best}")
```

## ğŸ§ª **Part 5: Basic Testing with Assert**

### **What is Assert?**

`assert` is a simple way to check if something is true. If it's not, Python raises an `AssertionError`.

```python
# Basic assert syntax
assert condition, "Error message if condition is False"

# Examples
assert 1 + 1 == 2, "Math is broken!"  # Passes (no output)
assert 2 + 2 == 5, "This will fail!"  # Raises AssertionError
```

### **Using Assert for Testing**

```python
# Function to test
def add(a, b):
    return a + b

# Test the function
assert add(2, 3) == 5, "add(2, 3) should be 5"
assert add(-1, 1) == 0, "add(-1, 1) should be 0"
assert add(0, 0) == 0, "add(0, 0) should be 0"

print("All tests passed! âœ…")
```

### **Testing with Dictionaries**

