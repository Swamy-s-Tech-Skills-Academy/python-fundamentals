## üîÑ **Part 1: For Loops & range()**

### **What is a For Loop?**

A `for` loop repeats code a specific number of times. It's perfect when you know exactly how many times you want to repeat something.

### **Basic For Loop with range()**

```python
# Print "Hello" 5 times
for i in range(5):
    print("Hello!")

# Output:
# Hello!
# Hello!
# Hello!
# Hello!
# Hello!
```

### **Understanding range()**

The `range()` function generates a sequence of numbers:

```python
# range(stop) - 0 to stop-1
for i in range(5):
    print(i)  # Prints: 0, 1, 2, 3, 4

# range(start, stop) - start to stop-1
for i in range(1, 6):
    print(i)  # Prints: 1, 2, 3, 4, 5

# range(start, stop, step) - with custom step
for i in range(0, 10, 2):
    print(i)  # Prints: 0, 2, 4, 6, 8

# Counting backwards
for i in range(5, 0, -1):
    print(i)  # Prints: 5, 4, 3, 2, 1
```

### **Common range() Patterns**

```python
# Count from 1 to 10
for num in range(1, 11):
    print(num)

# Sum numbers 1 to 100
total = 0
for num in range(1, 101):
    total += num
print(f"Sum: {total}")  # Output: Sum: 5050

# Multiplication table
number = 7
for i in range(1, 11):
    print(f"{number} x {i} = {number * i}")
```

### **The Loop Variable**

The variable (commonly `i`) takes each value from the sequence:

```python
for count in range(3):
    print(f"Count is: {count}")

# Output:
# Count is: 0
# Count is: 1
# Count is: 2
```

## üìù **Part 2: Iterating Over Sequences**

### **Looping Through Strings**

Strings are sequences of characters. You can loop through each character:

```python
name = "Python"
for letter in name:
    print(letter)

# Output:
# P
# y
# t
# h
# o
# n
```

### **Looping Through Lists**

Lists are collections of items (we'll learn more in Session 7):

```python
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I like {fruit}!")

# Output:
# I like apple!
# I like banana!
# I like cherry!
```

### **Using enumerate() for Index + Value**

Sometimes you need both the position and the value:

```python
colors = ["red", "green", "blue"]

for index, color in enumerate(colors):
    print(f"{index}: {color}")

# Output:
# 0: red
# 1: green
# 2: blue

# Start counting from 1
for index, color in enumerate(colors, start=1):
    print(f"{index}. {color}")

# Output:
# 1. red
# 2. green
# 3. blue
```

### **Looping with Index Using range() and len()**

```python
fruits = ["apple", "banana", "cherry"]

for i in range(len(fruits)):
    print(f"Item {i}: {fruits[i]}")

# Output:
# Item 0: apple
# Item 1: banana
# Item 2: cherry
```

## ‚è≥ **Part 3: While Loops**

### **What is a While Loop?**

A `while` loop repeats code **as long as a condition is True**. It's perfect when you don't know exactly how many times you need to repeat.

### **Basic While Loop**

```python
count = 0
while count < 5:
    print(f"Count: {count}")
    count += 1  # Don't forget to update!

# Output:
# Count: 0
# Count: 1
# Count: 2
# Count: 3
# Count: 4
```

### **‚ö†Ô∏è Infinite Loops (Danger!)**

If the condition never becomes `False`, the loop runs forever:

```python
# DON'T DO THIS - infinite loop!
# count = 0
# while count < 5:
#     print(count)
#     # Forgot count += 1 !

# Press Ctrl+C to stop an infinite loop
```

### **While Loop with User Input**

Perfect for menus and repeated prompts:

```python
# Keep asking until valid input
while True:
    answer = input("Enter 'yes' or 'no': ").lower()
    if answer in ["yes", "no"]:
        break  # Exit the loop
    print("Invalid input! Try again.")

print(f"You entered: {answer}")
```

### **Countdown Example**

```python
import time

countdown = 5
while countdown > 0:
    print(countdown)
    countdown -= 1
    time.sleep(1)  # Wait 1 second

print("üöÄ Liftoff!")
```

### **While vs For: When to Use Each**

| Use `for` when...                    | Use `while` when...                    |
| ------------------------------------ | -------------------------------------- |
| You know how many iterations         | You don't know how many iterations     |
| Iterating over a sequence            | Waiting for a condition to change      |
| Processing each item in a collection | User input validation                  |
| Counting a specific number of times  | Running until something happens        |

## üéÆ **Part 4: Loop Controls**

### **break - Exit the Loop Immediately**

`break` stops the loop completely:

```python
# Find first even number
for num in range(1, 10):
    if num % 2 == 0:
        print(f"First even: {num}")
        break  # Exit the loop
    print(f"Checking {num}...")

# Output:
# Checking 1...
# First even: 2
```

### **continue - Skip to Next Iteration**

`continue` skips the rest of the current iteration:

```python
# Print only odd numbers
for num in range(1, 10):
    if num % 2 == 0:
        continue  # Skip even numbers
    print(num)

# Output: 1, 3, 5, 7, 9
```

### **pass - Do Nothing (Placeholder)**

`pass` is a placeholder when you need a statement but don't want to do anything:

```python
# Placeholder for future code
for i in range(5):
    if i == 3:
        pass  # TODO: handle this case later
    else:
        print(i)
```

### **else with Loops**

Python loops can have an `else` block that runs when the loop completes normally (without `break`):

```python
# Search for a value
numbers = [1, 3, 5, 7, 9]
target = 6

for num in numbers:
    if num == target:
        print(f"Found {target}!")
        break
else:
    print(f"{target} not found in list.")

# Output: 6 not found in list.
```

### ‚úÖ **Progress Check: Loop Controls**

Make sure you understand:

- [ ] `break` exits the loop completely
- [ ] `continue` skips to the next iteration
- [ ] `pass` is a do-nothing placeholder
- [ ] `else` runs if loop completes without break

## üî≤ **Part 5: Nested Loops**

### **What are Nested Loops?**

A loop inside another loop. The inner loop runs completely for each iteration of the outer loop.

### **Basic Nested Loop**

```python
for i in range(3):
    for j in range(3):
        print(f"({i}, {j})")

# Output:
# (0, 0)
# (0, 1)
# (0, 2)
# (1, 0)
# (1, 1)
# (1, 2)
# (2, 0)
# (2, 1)
# (2, 2)
```

### **Creating a Grid**

```python
# Print a 3x3 grid
for row in range(3):
    for col in range(3):
        print("*", end=" ")
    print()  # New line after each row

# Output:
# * * *
# * * *
# * * *
```

### **Multiplication Table**

```python
print("Multiplication Table (1-5)\n")
print("   ", end="")
for i in range(1, 6):
    print(f"{i:4}", end="")
print("\n" + "-" * 25)

for i in range(1, 6):
    print(f"{i} |", end="")
    for j in range(1, 6):
        print(f"{i*j:4}", end="")
    print()

# Output:
#      1   2   3   4   5
# -------------------------
# 1 |  1   2   3   4   5
# 2 |  2   4   6   8  10
# 3 |  3   6   9  12  15
# 4 |  4   8  12  16  20
# 5 |  5  10  15  20  25
```

### **Pattern Printing**

```python
# Right triangle
rows = 5
for i in range(1, rows + 1):
    print("*" * i)

# Output:
# *
# **
# ***
# ****
# *****

# Pyramid
for i in range(1, rows + 1):
    spaces = " " * (rows - i)
    stars = "*" * (2 * i - 1)
    print(spaces + stars)

# Output:
#     *
#    ***
#   *****
#  *******
# *********
```

### **Script 3: 03_fizzbuzz.py**

```python
# Filename: src/L1/S5/03_fizzbuzz.py

"""
üéØ FizzBuzz Challenge

Classic programming challenge:
- Print numbers 1 to n
- For multiples of 3, print "Fizz"
- For multiples of 5, print "Buzz"
- For multiples of both, print "FizzBuzz"
"""

print("=" * 40)
print("üéØ FIZZBUZZ CHALLENGE")
print("=" * 40)

n = int(input("\nEnter a number (try 30): "))

print(f"\nFizzBuzz from 1 to {n}:")
print("-" * 30)

for num in range(1, n + 1):
    if num % 3 == 0 and num % 5 == 0:
        print("FizzBuzz")
    elif num % 3 == 0:
        print("Fizz")
    elif num % 5 == 0:
        print("Buzz")
    else:
        print(num)

print("-" * 30)
print("‚ú® Challenge complete!")

# Bonus: Pattern printing
print("\n" + "=" * 40)
print("üé® PATTERN PRINTING")
print("=" * 40)

rows = 5

print("\n--- Right Triangle ---")
for i in range(1, rows + 1):
    print("*" * i)

print("\n--- Number Triangle ---")
for i in range(1, rows + 1):
    for j in range(1, i + 1):
        print(j, end=" ")
    print()

print("\n--- Pyramid ---")
for i in range(1, rows + 1):
    spaces = " " * (rows - i)
    stars = "*" * (2 * i - 1)
    print(spaces + stars)
```

## üîß **Troubleshooting**

### **Common Issues:**

#### **Infinite while loop:**

```python
# Wrong - never ends!
count = 0
while count < 5:
    print(count)  # Forgot count += 1

# Right
count = 0
while count < 5:
    print(count)
    count += 1  # Don't forget!
```

#### **Off-by-one errors:**

```python
# Prints 0-4 (not 1-5)
for i in range(5):
    print(i)

# Prints 1-5
for i in range(1, 6):
    print(i)
```
