## ğŸ“ **Part 1: Python Indentation**

### **Why Does Python Use Indentation?**

Unlike many programming languages that use braces `{}` to define code blocks, Python uses **indentation** (spaces or tabs). This makes code cleaner and forces good formatting habits!

### **The Rule: Consistent Indentation**

```python
# âœ… CORRECT - 4 spaces (recommended)
if True:
    print("This is indented")
    print("So is this")

# âŒ WRONG - Inconsistent indentation
if True:
    print("This is indented")
  print("This will cause an error!")  # IndentationError!
```

### **Standard Convention: 4 Spaces**

```python
# Python community standard: 4 spaces per indentation level
if condition:
    # First level: 4 spaces
    do_something()
    if another_condition:
        # Second level: 8 spaces
        do_something_else()
```

### **Common Indentation Errors**

```python
# Error 1: Missing indentation
if True:
print("Error!")  # IndentationError: expected an indented block

# Error 2: Mixing tabs and spaces
if True:
    print("Using spaces")
    print("Using tab")  # TabError: inconsistent use of tabs and spaces

# Error 3: Unexpected indent
x = 5
    y = 10  # IndentationError: unexpected indent
```

VS Code automatically handles indentation! When you press Enter after a `:`, it adds 4 spaces automatically.

## ğŸ¤” **Part 2: Conditional Statements**

### **What are Conditional Statements?**

Conditional statements let your program make decisions. Based on whether a condition is `True` or `False`, different code runs.

### **The `if` Statement**

```python
age = 18

if age >= 18:
    print("You are an adult!")
    print("You can vote!")
```

**How it works:**

1. Python evaluates `age >= 18`
2. If `True`, the indented code runs
3. If `False`, the indented code is skipped

### **The `if-else` Statement**

```python
age = 15

if age >= 18:
    print("You are an adult!")
else:
    print("You are a minor.")
```

**Either the `if` block OR the `else` block runs â€” never both!**

### **The `if-elif-else` Statement**

When you have multiple conditions, use `elif` (else if):

```python
score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
elif score >= 60:
    print("Grade: D")
else:
    print("Grade: F")
```

**Important:** Python checks conditions from top to bottom and stops at the first `True` condition.

### **Multiple Conditions Example**

```python
temperature = 25

if temperature < 0:
    print("Freezing! â„ï¸")
elif temperature < 10:
    print("Cold! ğŸ¥¶")
elif temperature < 20:
    print("Cool ğŸŒ¤ï¸")
elif temperature < 30:
    print("Warm â˜€ï¸")
else:
    print("Hot! ğŸ”¥")
```

### **Nested Conditionals**

You can put `if` statements inside other `if` statements:

```python
has_ticket = True
age = 15

if has_ticket:
    if age >= 18:
        print("Welcome to the adult section!")
    else:
        print("Welcome to the family section!")
else:
    print("Please buy a ticket first.")
```

## ğŸ”— **Part 3: Boolean Logic**

### **Combining Conditions**

Sometimes you need to check multiple conditions together. Python provides three Boolean operators:

| Operator | Description                      | Example                     |
| -------- | -------------------------------- | --------------------------- |
| `and`    | Both conditions must be True     | `age >= 18 and has_id`      |
| `or`     | At least one condition is True   | `is_student or is_senior`   |
| `not`    | Reverses the Boolean value       | `not is_banned`             |

### **The `and` Operator**

Both conditions must be `True`:

```python
age = 25
has_license = True

if age >= 18 and has_license:
    print("You can drive!")
else:
    print("You cannot drive.")

# Truth table for 'and':
# True and True   = True
# True and False  = False
# False and True  = False
# False and False = False
```

### **The `or` Operator**

At least one condition must be `True`:

```python
is_weekend = True
is_holiday = False

if is_weekend or is_holiday:
    print("You can sleep in! ğŸ˜´")
else:
    print("Time to wake up early! â°")

# Truth table for 'or':
# True or True   = True
# True or False  = True
# False or True  = True
# False or False = False
```

### **The `not` Operator**

Reverses `True` to `False` and vice versa:

```python
is_raining = False

if not is_raining:
    print("Let's go outside! ğŸŒ")
else:
    print("Better stay inside. ğŸŒ§ï¸")

# not True  = False
# not False = True
```

### **Combining Multiple Operators**

```python
age = 25
is_student = True
has_coupon = False

# Complex condition
if (age < 30 and is_student) or has_coupon:
    print("You get a discount!")
else:
    print("Regular price.")
```

**Use parentheses to make complex conditions clear!**

### **Practical Example: Login Validation**

```python

password = input("Enter password: ")

correct_user = "admin"
correct_pass = "secret123"

if username == correct_user and password == correct_pass:
    print("âœ… Login successful!")

    print("âŒ Wrong password")
```

## ğŸ“¦ **Part 4: Introduction to Modules**

### **Built-in Functions vs Modules**

**Built-in functions** are always available (no import needed):

```python
# These work immediately
print("Hello")
len("Hello")
type(42)
input("Name: ")
```

**Modules** are collections of additional functions that need to be imported:

```python
# Must import before using
import math
print(math.sqrt(16))  # Output: 4.0
```

### **Why Modules?**

- Python keeps the core language small and fast
- Additional functionality is organized into modules
- You only load what you need

### **Importing Modules**

#### Method 1: Import the whole module

```python
import math

print(math.pi)        # Output: 3.141592653589793
print(math.sqrt(25))  # Output: 5.0
print(math.floor(3.7)) # Output: 3
```

#### Method 2: Import specific items

```python
from math import pi, sqrt

print(pi)        # Output: 3.141592653589793
print(sqrt(25))  # Output: 5.0 (no math. prefix needed!)
```

#### Method 3: Import with alias

```python
import math as m

print(m.pi)       # Output: 3.141592653589793
print(m.sqrt(25)) # Output: 5.0
print(m.floor(3.7)) # Output: 3
```

### **Common Built-in Modules**

| Module     | Purpose                        | Example Functions              |
| ---------- | ------------------------------ | ------------------------------ |
| `math`     | Mathematical functions         | `sqrt()`, `pi`, `floor()`      |
| `random`   | Random number generation       | `randint()`, `choice()`        |
| `datetime` | Date and time handling         | `now()`, `date()`              |
| `os`       | Operating system interaction   | `getcwd()`, `listdir()`        |

## ğŸ² **Part 5: The `random` Module**

### **What is the `random` Module?**

The `random` module lets you generate random numbers and make random choices â€” perfect for games!

### **Generating Random Integers**

```python
import random

# Random integer between 1 and 10 (inclusive)
number = random.randint(1, 10)
print(f"Random number: {number}")

# Roll a dice (1-6)
dice = random.randint(1, 6)
print(f"You rolled: {dice}")
```

### **Random Floating-Point Numbers**

```python
import random

# Random float between 0.0 and 1.0
decimal = random.random()
print(f"Random decimal: {decimal}")

# Random float in a range
price = random.uniform(10.0, 50.0)
print(f"Random price: ${price:.2f}")
```

### **Making Random Choices**

```python
import random

# Choose from a list
colors = ["red", "green", "blue", "yellow"]
chosen = random.choice(colors)
print(f"Random color: {chosen}")

# Choose multiple items (no repeats)
winners = random.sample(colors, 2)
print(f"Winners: {winners}")

# Choose multiple items (with possible repeats)
picks = random.choices(colors, k=3)
print(f"Picks: {picks}")
```

### **Shuffling a List**

```python
import random

cards = ["Ace", "King", "Queen", "Jack", "10"]
print(f"Original: {cards}")

random.shuffle(cards)  # Shuffles in place!
print(f"Shuffled: {cards}")
```

### **Setting a Seed (Reproducible Results)**

```python
import random

# Same seed = same sequence of "random" numbers
random.seed(42)
print(random.randint(1, 100))  # Always 82
print(random.randint(1, 100))  # Always 15
```

## ğŸ”§ **Troubleshooting**

### **Common Issues:**

#### **IndentationError:**

```python
# Wrong
if True:
print("No indent!")  # Error!

# Right
if True:
    print("Properly indented!")
```

#### **Forgetting the colon:**

```python
# Wrong
if age >= 18  # Missing colon!
    print("Adult")

# Right
if age >= 18:
    print("Adult")
```

#### **Using `=` instead of `==`:**

```python
# Wrong (assignment, not comparison)
if x = 5:  # SyntaxError!

# Right (comparison)
if x == 5:
    print("x is 5")
```
